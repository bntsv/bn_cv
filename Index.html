wordSoup

the idea here is to create a webpage that generates wordsearch puzzles. Users will be able to define size and words to be used in the puzzle. No UX/UI is implemented yet, but the js can fill in a set of words at randomly selected positions in an array (the puzzle). If a word cant fit it is added in a skippedWords array (logged last in console). Upon page refresh new positions of the words are generated. As part of the concept the title of the puzzle is always in the middle of the array, which complicates the task a bit.
